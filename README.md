## 📖 نبذة عن المشروع

**GT-SARARIM** هو تطبيق تعليمي ترفيهي مصمم خصيصاً للطفل العربي المسلم، يجمع بين التعلم والمرح في بيئة آمنة **تعمل بالكامل دون الحاجة إلى الإنترنت**. يهدف التطبيق إلى غرس القيم الإسلامية وتعزيز المعرفة العلمية واللغوية من خلال:

- 📖 **قصص مشكّلة بالكامل** (الأنبياء، الأخلاق، العلماء، العلوم)
- 🧠 **ألغاز ذكية** وأسئلة تفاعلية
- 🎨 **أنشطة متنوعة** (مسابقة معلومات، اختبار صور، وصل النقاط، تلوين)
- 🏆 **نظام تحفيز** (نقاط، مستويات، شهادات إنجاز)
- 👨‍👩‍👧 **تحكم كامل للوالدين** (إضافة محتوى، وقت لعب، قفل أقسام، نسخ احتياطي)
- 🌙 **وضع فاتح/داكن** يكتشف تلقائياً إعدادات النظام

> **فكرة فريدة**: يمكن للوالدين إضافة قصصهم وأسئلتهم وألغازهم الخاصة **دون انتظار تحديثات التطبيق** — كل ذلك عبر لوحة تحكم سهلة الاستخدام.

## 🛡️ لوحة تحكم الوالدين

- **الدخول:** كلمة المرور الافتراضية `admin` (يمكن تغييرها بعد الدخول من الإعدادات)
- **إدارة الأبناء:** إضافة حسابات، صور، كلمات مرور، وقت لعب يومي
- **إدارة المحتوى:** إضافة/تعديل/حذف القصص، الأسئلة، الألغاز، الفئات
- **الأمان:** قفل أقسام، تراجع عن الحذف (10 ثوانٍ)، سؤال استرداد كلمة المرور
- **النسخ الاحتياطي:** تصدير واستيراد البيانات كاملة (JSON)

> ⚠️ **هام:** كلمة المرور الافتراضية للوالدين هي `admin`. يُنصح بتغييرها فور الدخول الأول من خلال إعدادات لوحة التحكم.

## 🎨 نظرة على المحتوى

### قصص (8+ قصة)
- **قصص الأنبياء** (نوح، يوسف)
- **الأخلاق والقيم** (الصدق، بر الوالدين)
- **علماء المسلمين** (الخوارزمي، ابن سينا)
- **قصص العلوم** (الفضاء، الحيوانات)

### أنشطة
| النشاط | الوصف |
|--------|-------|
| 🧠 مسابقة المعلومات | أسئلة متعددة المجالات مع خلط تلقائي للخيارات |
| 🖼 اختبار الصور | تعرّف على الشكل الصحيح من خيارات SVG |
| ✏️ وصل النقاط | 6 أشكال SVG تفاعلية (نجمة، بيت، قمر، سمكة، زهرة، جمل) |
| 🎨 التلوين الإبداعي | 6 صور SVG للتلوين مع لوحة ألوان |

### نظام المستويات
| المستوى | العمر | النقاط المطلوبة |
|---------|-------|-----------------|
| 🌱 تمهيدي | 4-6 سنوات | 0 |
| 📚 مبتدئ | 6-8 سنوات | 500 |
| ⭐ متوسط | 8-10 سنوات | 1500 |
| 🏆 متقدم | 10-12 سنة | 3000 |

## 🛠️ بناء التطبيق بنفسك

### بناء لينكس (AppImage + deb)
```bash
npm run pack:linux
# النتيجة في مجلد release/
```

### بناء أندرويد (APK)
```bash
npm run android:setup   # مرة واحدة
npm run android:debug   # APK تجريبي
# أو
npm run android:build   # APK نهائي (يتطلب توقيع)
```

### بناء نسخة الويب
```bash
npm run build
# محتويات مجلد dist/ هي الموقع الجاهز للنشر
```
