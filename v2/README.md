# GT-SARARIM v2.1 — سارة ريم عالم المعرفة والمرح

تطبيق تعليمي وترفيهي متكامل للطفل العربي المسلم — يعمل بالكامل بدون إنترنت.

## 🆕 مستجدات الإصدار v2.1

### إصلاحات
- ✅ إصلاح أيقونات وصل النقاط (كانت تظهر جمل متكرر)
- ✅ إصلاح تجاوب الشاشة على الهاتف (شريط التنقل السفلي لم يكن صحيحاً)
- ✅ إصلاح عرض المحتوى على الحاسوب (نطاق أكثر راحة)
- ✅ تحسين تجربة الألغاز: خيارات متعددة + صورة ضبابية تنكشف عند الإجابة

### محتوى جديد — المستوى المتقدم (expert)
- **8 ألغاز جديدة** للمستوى الخبير (رياضيات، فيزياء، منطق، إسلاميات)
- **8 أسئلة جديدة** للمستوى الخبير
- **3 قصص جديدة** للمستوى الخبير (العصر الذهبي، الاجتهاد، فيزياء الكون)
- **2 قصص جديدة** للمستوى المتوسط (ابن الهيثم، البيروني)
- **أسئلة المستوى المتوسط والمتقدم** (6 أسئلة إضافية)

### تحسينات الألغاز
- ميزة **4 خيارات متعددة** كمساعدة (10 نقاط عند الاستخدام)
- ميزة **الصورة الضبابية** تنكشف تدريجياً عند الإجابة الصحيحة
- مؤشر بصري واضح على الألغاز التي تحتوي صور أو تلميح

## التثبيت والتشغيل

### متطلبات التطوير
```bash
Node.js >= 18
npm >= 9
```

### تشغيل في وضع التطوير
```bash
npm install
npm run dev
```

### بناء وتحزيم
```bash
# Linux (AppImage + deb)
npm run pack:linux

# Windows (NSIS installer)
npm run pack:win

# Android APK
npm run android:setup
npm run android:build
```

### تشغيل Electron مباشرة
```bash
npm run build
npm run electron
```
