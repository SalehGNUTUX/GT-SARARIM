# GT-SARARIM v2.3.0 — سارة ريم عالم المعرفة والمرح

تطبيق تعليمي وترفيهي متكامل للطفل العربي المسلم — يعمل بالكامل بدون إنترنت.

## 🆕 مستجدات الإصدار v2.3.0

### إصلاحات
- ✅ **زر الرجوع أندرويد**: يعود إلى التبويب السابق؛ يعرض توست "اضغط مرتين للخروج" من الرئيسية
- ✅ **تكرار URL عند المشاركة**: دمج النص والرابط في حقل واحد لتفادي تكراره
- ✅ **أسماء الفئات عند استيراد نسخة قديمة**: تُعاد الأسماء العربية الصحيحة لجميع الفئات المدمجة تلقائياً
- ✅ **موضع الصفحة عند تغيير التبويب**: الصفحة تعود الآن إلى الأعلى تلقائياً عند التنقل بين الأقسام
- ✅ **الخط الافتراضي**: أصبح Noto Naskh Arabic (ترقية تلقائية لمستخدمي ubuntu-arabic)

## التثبيت والتشغيل

### متطلبات التطوير
```bash
Node.js >= 18
npm >= 9
Android Studio (للأندرويد فقط)
```

### تشغيل في وضع التطوير
```bash
npm install
npm run dev
```

### بناء وتحزيم
```bash
# Linux (AppImage + deb)
npm run pack:linux

# Windows (NSIS installer)
npm run pack:win

# Android APK
npm run android:sync
npm run android:build
```

### تشغيل Electron مباشرة
```bash
npm run build
npm run electron
```

## البنية التقنية
- **React 19** + TypeScript
- **Tailwind CSS v4** + shadcn/ui
- **Zustand** للحالة (3-layer: IndexedDB + localStorage + Filesystem)
- **Motion/React** للحركات والانتقالات
- **Electron 35** للتطبيق المكتبي
- **Capacitor 6** للأندرويد + `@capacitor/share`
- **Web Audio API** لتوليد أصوات الخلفية
